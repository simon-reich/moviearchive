Linux:

Frontend:

1. install node.js: 
sudo apt install nodejs

2. install nvm: 
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

3. install specific node.js version:
nvm ls-remote (shows all versions)
nvm install <version> 

4. install npm:
sudo apt install npm

5. install pnpm:
npm install -g pnpm

6. install dependencies:
-> go to folder with package.json
pnpm install

7. start vite:
npm run dev 


Backend:

1. download docker containers of ElasticSearch and PostgreSQL
-> go to backend
docker-compose up

2. create virtual python environment:
in backend: pipenv shell

3. install python libraries:
-> go to folder with requirements.txt
pipenv install -r requirements.txt 

4. set up database:
-> go to src folder
flask db init
flask db migrate
flask db upgrade

5. start flask backend:
-> in src folder:
flask run --debug
